# Rajesh VVS - Portfolio Website

A modern, interactive portfolio showcasing my projects, skills, and achievements as a Computer Science Engineer and AI Builder.

## Features

- 3D animated background with video overlay
- Interactive project cards with tilt effects
- Smooth scroll animations
- Responsive design
- Modern glassmorphism UI

## Run Locally

**Prerequisites:** Node.js 18.x or higher

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your project images to `public/projects/`:
   - `ai-cofounder.jpg`
   - `ai-bias.jpg`
   - `swap-circle.jpg`

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

### Quick Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone)

### Manual Deploy

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click "Add New Project"
4. Import your GitHub repository
5. Click "Deploy" (Vercel auto-detects Vite)

Your site will be live at `https://your-project-name.vercel.app`

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.

## Adding Your Project Images

1. Place your images in `public/projects/` folder
2. Name them exactly as:
   - `ai-cofounder.jpg` (or .png)
   - `ai-bias.jpg` (or .png)
   - `swap-circle.jpg` (or .png)
3. Recommended: 800x600px, under 500KB
