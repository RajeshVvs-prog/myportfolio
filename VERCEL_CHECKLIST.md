# Vercel Deployment Checklist ✅

## Before Deploying

- [ ] Add your 3 project images to `public/projects/`:
  - [ ] `ai-cofounder.jpg` or `.png`
  - [ ] `ai-bias.jpg` or `.png`
  - [ ] `swap-circle.jpg` or `.png`

- [ ] Test build locally:
  ```bash
  npm run build
  ```

- [ ] Preview the build:
  ```bash
  npm run preview
  ```

- [ ] Update personal links in `src/App.tsx` (if needed):
  - [ ] GitHub URL
  - [ ] LinkedIn URL
  - [ ] Email address
  - [ ] Social media links

## Deployment Steps

- [ ] Push code to GitHub
- [ ] Connect repository to Vercel
- [ ] Click Deploy
- [ ] Wait for build to complete (~2 min)
- [ ] Visit your live site!

## Post-Deployment

- [ ] Test all project links work
- [ ] Check images load correctly
- [ ] Test on mobile devices
- [ ] Share your portfolio URL!

## Your Portfolio Will Be Ready! 🚀

Live at: `https://your-project-name.vercel.app`
